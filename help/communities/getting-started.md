# AEM Communities 快速入门 {#getting-started-with-aem-communities}

AEM Communities允许公司从站点模板库中创建专门的品牌社区体验。 论坛、组、博客、文件共享、日历、活动信息源和扩展用户配置文件可以集成到您的网站中，以促进propect、客户和公司之间的交互。
