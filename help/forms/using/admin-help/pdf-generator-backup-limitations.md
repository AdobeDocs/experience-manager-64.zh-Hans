# PDF生成器备份限制{#pdf-generator-backup-limitations}

无法备份PDF生成器用于转换文件的临时目录。 即使服务将正确恢复，数据仍可能丢失，因为PDF生成器会按设置的时间间隔审阅和清除临时目录的内容。
