# 检查列表{#checklists}

以下检查列表重点介绍了一些主要问题 — 请阅读与测试相关的其他部分以获取完整的详细信息。

| 检查 | 响应 | 操作 | 所有者 | 到期日期 |
|---|---|---|---|---|
| 是否已建立明确、定期的报告机制？ |  |  |  |  |
| 状态会议是否涉及关键参与者（来自测试团队）？ |  |  |  |  |
| 是否明确定义了测试责任（特别是对于客户）？ |  |  |  |  |
| 客户是否具有履行测试承诺所需的经验？ |  |  |  |  |
| 是否已选择所有工具（测试和跟踪）？ |  |  |  |  |
| 是否已安装上述工具并根据需要对用户（主要是项目团队）进行培训？ |  |  |  |  |
| 是否制定了明确、全面的测试计划？ |  |  |  |  |
| 测试计划是否已由相关方审查？ |  |  |  |  |
| 验收测试是否已明确定义？ |  |  |  |  |
| 各方是否正式接受验收测试？ |  |  |  |  |
